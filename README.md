# 项目功能简介

## 项目使用

1. vue3
   1. vue-router
   2. pinai
      1. pinia-plugin-persistedstate 持久化
   3. element-plus
   4. v-md-editor(插件)
2. typtScript
3. axios
4. less
5. koa2
6. mysql

## 基本功能

### 用户模块

1. 注册
2. 登录
3. 忘记密码
4. 退出登录
5. 上传头像
6. 上传个人背景
7. 编辑个人资料

### 文章模块

1. 发布文章
   1. 文章封面图片
2. 文章列表遍历
   1. 推荐 / 最新 / 热榜
   2. 浏览量
   3. 点赞
   4. 评论
3. 单文章展示
   1. 文章内容 vmd-editor
   2. 目录
   3. 点赞（仅单次点赞）
4. 评论

### 评论

1. 发布
   1. 一级
   2. 二级
2. 回复
3. 点赞
4. 删除
